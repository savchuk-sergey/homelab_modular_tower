## Why this workflow was cancelled

The mk0.10 SVG iteration produced technically correct but visually flat drawings
with no engineering judgment built in. Iterating on them did not unlock any design
decision that could not be read directly from `cad/config.py`. Physical prototyping
produces faster and more reliable validation than SVG drawing iteration for this
type of project.
