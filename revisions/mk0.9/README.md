# mk0.9 CAD Revision

mk0.9 is a modular architecture prototype. It converts the current tower direction into a four-module vertical platform with a shared draft interface.

mk0.9 is not the final mk1.0. The real mini PC dimensions are still unknown, so the mini PC tray, airflow duct, rear I/O cutouts, cable clearances and module height remain placeholder-based.

## Module Stack

From bottom to top:

- `01_base_module`
- `02_rpi_ssd_module`
- `03_mini_pc_placeholder_module`
- `04_roof_module`

Power distribution, router module, full rear service spine, DC UPS and final cable management are intentionally deferred.

## Implemented Scope

- Tower Module Interface v1 draft.
- Four corner M5 threaded rod zones.
- Top and bottom module interface features.
- Central airflow channel placeholder.
- Rear reserved zone for a future service spine.
- 120 mm bottom intake and 120 mm top exhaust placeholders.
- Raspberry Pi 3B, external SSD and mini PC bounding-box placeholders.
