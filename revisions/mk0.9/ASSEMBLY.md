# mk0.9 Assembly

1. Install TPU feet or temporary rubber feet.
2. Install the lower 120 mm intake fan in the base module.
3. Install the lower dust filter / filter tray.
4. Stack the RPi/SSD module onto the base module.
5. Install Raspberry Pi 3B on the standoffs.
6. Install the external SSD on the universal retainer area.
7. Stack the mini PC placeholder module.
8. Place the mini PC bounding-box placeholder for clearance review.
9. Stack the roof module.
10. Install the upper 120 mm exhaust fan.
11. Install the top guard/filter mesh.
12. Tie the stack with four M5 threaded rods, washers and nuts.

Each module should remain removable without dismantling the complete tower.
