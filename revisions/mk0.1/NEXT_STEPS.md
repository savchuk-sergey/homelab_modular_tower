# Roadmap For mk0.2

mk0.2 should start from mk0.1 in a new branch and must not rewrite mk0.1 history.

Planned work:

- Add tray locks/latches.
- Improve tray handles.
- Add real cable management paths.
- Add cable tie holes and strap slots where needed.
- Lighten side panels while preserving stiffness.
- Optimize airflow path.
- Improve Mini PC duct interface.
- Validate and improve washer pockets.
- Validate and improve nut pockets.
- Measure real Mini PC dimensions.
- Measure real MikroTik hAP ax2 dimensions.
- Measure real Raspberry Pi dimensions and standoff pattern.
- Measure real Kingston XS2000 dimensions and cable bend radius.
- Check collisions between trays, rails, rods, panels, and rear service spine.
- Validate metal rail loading and decide whether flat bar is enough.
- Validate all M3/M4/M5 fastener holes.
- Replace power bus placeholder zones with selected connector footprints.
- Add real strain relief for power and USB-C cables.
