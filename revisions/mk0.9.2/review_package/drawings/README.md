# mk0.9.2 drawing sheets

This folder contains generated engineering drawing sheets for the active
`mk0.9.2` CAD model.

Included:

- `assembly.png` - main assembly drawing sheet.
- `parts/*.png` - drawing sheets for non-legacy parts from
  `EXPORT_CATEGORIES`.

Excluded:

- `legacy` export-category helpers.

These drawings are generated review artifacts. CadQuery source remains the
source of truth.
