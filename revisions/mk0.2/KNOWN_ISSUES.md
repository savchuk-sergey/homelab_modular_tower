# Known Issues For mk0.2

mk0.2 improves service geometry, but it is still an engineering CAD revision that requires physical validation.

Known issues:

- Check grip comfort of the recessed carriage handle on a physical prototype.
- Check whether the tray front flange cracks when pulling a loaded module.
- Check PETG carriage clearances on the metal guide rails.
- Check whether the side panels warp during printing.
- Check whether side-panel ventilation worsens the intended vertical airflow.
- Check side-panel compatibility with the future Mini PC duct.
- Check side-panel screw access with the tower fully assembled.
- Confirm real M3 screw, insert, TPU plug, latch, or magnet dimensions before finalizing the lock boss.
- Confirm side-panel mounting-hole alignment against the real corner blocks and frame.
- Run thermal testing with all modules installed before treating the vent pattern as stable.
