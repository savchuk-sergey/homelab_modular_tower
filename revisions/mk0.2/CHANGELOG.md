# Changelog For mk0.2

Date: 2026-06-27

Changes relative to mk0.1:

- Changed `CURRENT_REVISION` to `mk0.2`.
- Added parametric carriage front plate, recessed handle pocket, reinforcement ribs, and future lock boss dimensions.
- Replaced the old protruding tray handle with a standardized recessed front service interface.
- Applied the same carriage handle path to all tray factories through `create_tray`.
- Added parametric side-panel dimensions for thickness, frame, ribs, vents, mounting bosses, clearance, and Mini PC duct keep-out.
- Reworked side panels as removable service covers with perimeter frame, X-style ribs, M3 mounting bosses, and controlled ventilation.
- Added mk0.2 revision documentation and known issues.
