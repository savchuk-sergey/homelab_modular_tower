# pre_mk012 CAD Legacy Archive

This directory contains CAD and CAD-pipeline files that were active before the
mk0.12 MVP-2M pre-CAD implementation cleanup.

These files are preserved for engineering history only.  They are not part of
the active mk0.12 CAD implementation path.

Active mk0.12 pre-CAD files live in:

- `cad/__init__.py`
- `cad/config.py`
- `scripts/validate_mk012_spec.py`

Do not import files from this archive into the active CAD path without an
explicit revision decision.
